📌 ##Sobre o Projeto

Este software foi desenvolvido como parte do Desafio 4.0 da disciplina de Estrutura de Dados. O objetivo é enfrentar a insegurança alimentar em São Luís, Maranhão, focando em comunidades vulneráveis, palafitas e ocupações irregulares, como as áreas do Rio Anil e Bacanga.
